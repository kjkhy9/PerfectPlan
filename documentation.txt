Folder structure of important folders in backend
backend/
│
├── controllers/
│     authcontroller.js
│     groupcontroller.js
│     eventcontroller.js
│
├── services/
│     authservice.js
│     groupservice.js
│     eventservice.js
│
├── routes/
│     authroutes.js
│     grouproutes.js
│     eventroutes.js
│
├── models/
│     user.js
│     group.js
│     event.js
│
├── db.js
├── server.js
└── .env  (Not included in GitHub)

All important logic for authentication of users, events, and group is in the services folder

dependency order 

Routes --> Controllers --> Services --> models

3 different MongoDb tables 
User, Group, and event

The frontend React app communicates with the backend through api calls

db.js is what connects the system to MongoDB

server.js is what connects all of the controllers, routes, services, and models to gether to send and recieve information to and from the frontend
